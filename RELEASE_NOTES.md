# 发布说明

## 1.0.0

首个黄金专用稳定版，围绕 `XAUUSD / M1` 完成：

- 黄金专用策略 `xau_m1_momentum`
- MT5 连接、下单、回测、风控
- 日内单方向开关
- 手工新闻黑窗与 MT5 自动新闻黑窗
- 中文配置说明、中文代码注释
- `exe` 打包支持

## 1.0.1

在 `1.0.0` 基础上扩展为多品种系统，重点新增 `BTCUSD / M15`：

- 新增 `btc_m15_regime` 专业策略
- 新增多品种切换启动器与图形界面启动器
- 新增 GUI 配置副本机制，不覆盖原始中文备注配置
- 新增 GUI 打开报表目录、日志目录、启动后自动关闭开关
- 新增本地运行日志 `logs/mt5-quant.log`
- 新增 `optimize-btc` 参数优化入口
- 新增 BTC 参数优化模板与中文回测报告模板
- 新增 `optimize-btc --workers` 并行提速能力
- 新增 `diagnose-signals` 信号诊断模式
- 新增 Python 启动前财经日历文件硬校验
- 重写 `ExportEconomicCalendar.mq5`
  - 默认只导出 `USD` 高影响新闻
  - 追加导出成功 / 失败状态日志
  - 增加启动自检提示
