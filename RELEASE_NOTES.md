# 发布说明

## 1.0.0

这是第一版只交易黄金的稳定版本，核心方向是 `XAUUSD / M1` 专用化。

### 主要内容

- 完成 `XAUUSD / M1` 专用策略
- 完成 MT5 连接、风控、下单、回测
- 完成交易时段过滤、日内熔断、连续亏损熔断
- 完成日内单方向开关
- 完成手工新闻黑窗
- 完成 MT5 自带经济日历自动黑窗
- 完成中文配置说明与中文代码注释
- 完成 `exe` 打包支持

## 1.0.1

这是在 `1.0.0` 基础上的策略扩展版本，重点加入 `BTCUSD` 专用品种策略。

### 主要内容

- 项目版本升级到 `1.0.1`
- 新增 `btc_m15_regime` 策略
- 新增 `BTCUSD / M15` 示例配置
- BTC 策略加入趋势强度、突破确认、成交量确认、ATR 动态止损
- 将 BTC 策略接入现有 CLI、回测、实盘、报表和风控系统

### BTC 策略逻辑概览

- 使用 `EMA Fast / EMA Slow` 识别趋势方向
- 使用 `ADX` 判断趋势是否足够强
- 使用 `Donchian Breakout` 判断是否突破关键区间
- 使用 `RSI` 做动量确认
- 使用 `Volume Mean` 过滤低质量突破
- 使用 `ATR Stop + Reward/Risk` 管理止损止盈
